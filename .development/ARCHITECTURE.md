# Architecture Reference

Quick reference for navigating the Government Feed codebase.
For detailed documentation, see `docs/architecture.md`.

## Layer Overview

| Layer | Package | Purpose |
|-------|---------|---------|
| **Core** | `backend/src/core/` | Domain entities, repository interfaces |
| **Infrastructure** | `backend/src/infrastructure/` | Database, feed parser, AI service, settings |
| **API** | `backend/src/api/` | FastAPI endpoints, Pydantic schemas |
| **Shared** | `shared/logging/` | Cross-cutting logging infrastructure |

**Dependency rule**: API -> Infrastructure -> Core. Core has no knowledge of Infrastructure or API.
